# Visual Prompts

## Theme Direction

- Apple inspired
- Glassmorphism
- Dark
- Blue
- Premium
- Minimal
- Recruiter friendly

## banner.png Prompt

Create a premium GitHub profile banner for a senior iOS engineer. Use an Apple-inspired dark glassmorphism style with deep navy gradients, cobalt highlights, soft blurred panels, subtle depth, and minimal technical motifs such as thin grid lines, abstract curves, or faint app-window shapes. Keep the center area clean for a name and title overlay. The composition should feel modern, elegant, and understated rather than busy. Use a wide banner format.

## social-preview.png Prompt

Create a dark premium social preview image for Siddharth Shah, Senior iOS Engineer. Use a glassmorphism card layout with blue lighting accents, Apple-inspired typography, and a clean hierarchy that highlights senior mobile engineering. Include subtle cues for Swift, UIKit, SwiftUI, clean architecture, and SDK development without clutter. Keep it polished, minimal, and highly legible in a 1200x630 social preview format.

## Repository Social Card Prompt

Create a repository social card for a premium GitHub profile repository. Use the same Apple-inspired dark blue glassmorphism language, but make the layout tighter and more product-focused. Include the repository name, a concise positioning statement, and a restrained set of visual accents. Keep it elegant, minimal, and ready to be used as a social share image or GitHub repository card.

## Notes

- Leave safe margins for cropping.
- Keep all text high contrast.
- Avoid neon purple and avoid playful illustration styles.
- Prefer crisp editorial spacing over dense composition.
